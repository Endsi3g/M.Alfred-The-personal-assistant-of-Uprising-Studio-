# Contributing to Project B.E.L.L. (Alfred)

Master Bell, we welcome your contributions to Alfred's tactical arsenal. To maintain the highest standards of the Batcave, please follow these guidelines.

## 🛡️ Protocol for Contributions

1. **Fork the Repository**: Create your own tactical branch.
2. **Follow the Style**: Align your code with the professional, discreet, and efficient standards of Alfred.
3. **Test Your Missions**: No code enters the root without passing `diagnostics.py` and `scripts/test_system.py`.
4. **Issue First**: Please open an issue (a "Bat-Signal") before starting major refactors.

## ⚔️ Skill Contributions

If you are adding a new skill to the `awesome_skills` catalog:

- Ensure it follows the `SKILL.md` format.
- Run `python modules/awesome_skills/generate_index.py` after your changes.

## 📜 Code of Conduct

Respect the cowl. Be professional, discreet, and helpful to the community.
